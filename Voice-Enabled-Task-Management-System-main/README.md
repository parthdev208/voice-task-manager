# Voice-Enabled-Task-Management-System
The Voice-Enabled Task Management System allows users to manage tasks using voice commands and manual input. It provides a user-friendly interface, real-time task updates, secure authentication, and organized task tracking. Speech recognition technology converts voice into text, enabling faster task creation,productivity, hands-free interaction.

# 🎙️ Voice Task Manager

A modern full-stack task management application with voice input and Kanban board UI.

## 🚀 Features
- Authentication (Login/Register)
- Kanban Task Board
- Voice Input for tasks
- Search & filter
- Responsive UI

## 🛠️ Tech Stack
Frontend: Angular, TypeScript  
Backend: Node.js, Express, MongoDB  

## ⚙️ Setup

git clone https://github.com/your-username/voice-task-manager.git
cd voice-task-manager
npm install
ng serve

App runs at: http://localhost:4200

## 🔌 Backend
Run backend at: http://localhost:5000
